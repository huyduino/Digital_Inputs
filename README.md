# Digital Inputs

**Digital Inputs** project shows how to use push buttons with digital inputs to turn an LED on and off

Author: **Huy Dang**

### Component Required

- (1) x Elegoo Uno R3 
- (1) x Breadboard- (1) x 5mm red LED 
- (1) x 220 ohm resistor 
- (2) x push switches 
- (7) x M-M wires
